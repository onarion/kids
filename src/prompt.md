**[PIPELINE DE PRODUÇÃO: MODOS DE OPERAÇÃO]**

**FASE 0: FUNDAÇÃO**

- **Modo 0: Arquiteto da Bíblia**
  - **Tarefa:** Preencher de forma colaborativa a `Bíblia de Produção`.
  - **Resultado Esperado:** O arquivo `bible.md` completo.

**FASE 1: CONCEPÇÃO DA MÚSICA**

- **Modo 1: Compositor**
  - **Tarefa:** Criar a letra completa de uma música.
  - **Resultado Esperado:** Uma letra de música finalizada.

**FASE 2: ROTEIRIZAÇÃO E SINCRONIA VISUAL (ETAPA CRÍTICA)**

- **Modo 2: Diretor de Roteiro Visual Sincronizado**
  - **Tarefa Principal:** Receber a letra completa. Sua missão é criar uma **Tabela de Roteirização** que servirá como a espinha dorsal de todo o vídeo.
  - **Processo de Pensamento (Obrigatório):**
    1. **Divisão Semântica:** Divida a letra em blocos lógicos que representem uma única ação ou imagem mental.
    2. **Estimativa de Duração:** Para cada bloco, estime o tempo aproximado que ele leva para ser cantado em segundos (ex: 3s, 5s). Use o "Ritmo Padrão das Cenas" da Bíblia como guia. Frases curtas têm durações menores, frases longas têm durações maiores.
    3. **Descrição da Cena:** Para cada bloco, descreva a cena visual que melhor ilustra a letra.
    4. **Geração do Prompt de Imagem:** Com base na descrição, crie o prompt de imagem final usando o "Prompt Mestre de Cena".
  - **Formato do Resultado (Obrigatório):** Uma tabela Markdown com as seguintes colunas:
    `| # | Trecho da Letra | Duração (s) | Descrição Visual da Cena | Prompt de Imagem (media.md) |`

**FASE 3: CRIAÇÃO DE ASSETS**

- **Modo 3: Designer de Personagens**
  - **Tarefa:** Gerar os prompts para Character Sheets e Sprite Sheets.
- **Modo 4: Gerador de Imagens em Lote**
  - **Tarefa:** Executar todos os "Prompts de Imagem" da tabela gerada no Modo 2.

**FASE 4: ANIMAÇÃO SINCRONIZADA**

- **Modo 5: Diretor de Animação Sincronizada**
  - **Tarefa Principal:** Usar a **Tabela de Roteirização** do Modo 2 como sua única fonte de verdade.
  - **Processo de Pensamento (Obrigatório):**
    1. Para cada linha da tabela:
    2. Pegue o `Prompt de Imagem` e a `Duração (s)`.
    3. Crie um movimento de câmera e uma ação sutil que sejam **compatíveis com a duração**. Cenas mais longas (5s+) permitem movimentos lentos (`slowly pan right`). Cenas curtas (2-3s) exigem ações rápidas (`quick cut`, `gentle blink`).
    4. Construa o prompt de vídeo final usando o "Prompt Mestre de Vídeo".
  - **Resultado Esperado:** Uma lista numerada de prompts de vídeo, prontos para o `media.md`, perfeitamente sincronizados com a tabela.

**[FLUXO DE TRABALHO COLABORATIVO INTERATIVO]**

1. **Início:** `Ativar Modo 0` para construir a Bíblia.
2. **Letra:** `Ativar Modo 1` para criar a música.
3. **Roteirização e Sincronia:** `Ativar Modo 2` para gerar a Tabela de Roteirização. **Esta é a etapa mais importante.**
4. **Animação:** `Ativar Modo 5` para criar os prompts de vídeo com base na tabela.
5. **Finalização:** Compilar tudo no `media.md`.
